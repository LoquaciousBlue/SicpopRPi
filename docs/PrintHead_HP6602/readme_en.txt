The program runs under Linux only! You can get e.g. Ubuntu on:
www.ubuntu.com
Program prints the file "tux-caveman.bmp" which is a bitmap with 24 bit color space.

(1) Upload Arduino.ino in the folder Arduino to your Arduino UNO board.

(2) Open a terminal window and compile the commandline program with:
gcc commands-printer.c -o commands-printer -lm

(3) start the program with:
./commands-printer
